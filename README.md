# Виртуальный Музей Морского Мусора | Онежское Поморье

Проект создан в рамках межрегионального конкурса **«Арктический Хакатон 2026»**. Трек: «Земля» (Проектное решение для ФГБУ «Национальный парк "Кенозерский"»).

Цель проекта — повысить осведомленность жителей региона и школьников Архангельской области о проблеме антропогенного загрязнения Белого моря и прибрежных территорий национального парка «Онежское Поморье».

## 🌟 Особенности проекта

- **Образовательный контент**: Информация о масштабах проблемы загрязнения морей пластиком и рыболовными снастями.
- **Фотогалерея**: Живописные пейзажи природы Онежского полуострова, демонстрирующие красоту региона, которую необходимо сохранить.
- **Интерактивные экспонаты**: Детальный разбор самых частых находок морского мусора на побережье (информация о материалах, вероятных причинах попадания в воду и уровне опасности).
- **Мини-игра «EcoSub: Очистка Океана»**: Интерактивная аркада на Canvas, в которой игрок управляет подводной лодкой (субмариной) и собирает мусор при помощи крюка, уклоняясь от препятствий.
- **Форма обратной связи**: Удобный модальный интерфейс для отправки отзывов и предложений пользователей с 5-звездочной системой оценки.

## 🛠 Технологии

Проект реализован с использованием базовых веб-технологий (без применения тяжелых фреймворков):
- **HTML5** — семантическая разметка.
- **CSS3** — стилизация, анимации, адаптивный дизайн для мобильных устройств.
- **JavaScript (Vanilla)** — логика интерактива (модальные окна, слайдер фотографий, обработка формы обратной связи, логика мини-игры).
- **Font Awesome** — векторные иконки интерфейса.
- **Google Fonts** — шрифты Montserrat и Nunito.

## 🚀 Как запустить

Проект представляет собой набор статических файлов. Для его просмотра не требуются сборщики (Node.js/Webpack).

1. Скачайте файлы проекта на свой компьютер.
2. Просто откройте файл `index.html` в любом современном веб-браузере (Chrome, Firefox, Safari и др.).

*Примечание: для корректного отображения шрифтов и иконок требуется подключение к интернету.*

## 📂 Основная структура файлов

- `index.html` — главная веб-страница, содержащая контент всего сайта (одностраничный лендинг).
- `style.css` — таблица стилей, описывающая визуальное оформление (дизайн, сетку, цвета, отступы).
- `script.js` — скрипт, содержащий всю программную логику интерфейса и игры.
- Папка `gallery/` — фотографии природы Онежского Поморья для секции-слайдера.
- Файлы `ex_*.png` — изображения виртуальных экспонатов (бутылка, буй, сапог, зонд и т.д).
- Прочие изображения (`hero.png`, `map.png` и др.) — графика для оформления сайта и мини-игры.

## 🏆 Контекст разработки

Разработано для участия в **Арктическом Хакатоне 2026** как информационный и обучающий ресурс поддержки национального парка «Онежское Поморье» и привлечения внимания к экологическим проблемам севера.
